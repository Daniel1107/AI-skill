## 使用方式

```bash
# 克隆到 Codex 技能目录
git clone https://github.com/Daniel1107/codex-skills.git $CODEX_HOME/skills

# 或作为子模块挂载
git submodule add https://github.com/Daniel1107/codex-skills.git .codex/skills
```
